CSS Box Model

  • "element" is the innermost part
  • "padding" is surrounding the element
  • "border" is surrounding the padding
  • "margin" is surrounding the border

CSS Colors

  • color:font-color
  • background-color:example
  • background:linear-gradient(example, example)

CSS Fonts

  • font-family:example, example, example
  • font-size:#px
  • font-weight:#00
  • font-style:example
  • text-transform:example
  • text-decoration:example

CSS Selectors

  • * - universal selector
  • h1 - element selector
  • .class-name - class selector
  • #id-name - id selector

CSS Displays

  • display:inline
  • display:block
  • display:inline-block
  • display:flex
  • display:none

CSS Positions

  • position:fixed
  • position:relative
  • position:absolute
  • position:static
  • position:sticky

CSS Media Queries

  • @media screen and (max-width: 992px) tablet
  • @media screen and (max-width: 768px) large phone
  • @media screen and (max-width: 576px) small phone

CSS Flexbox

  • display:flex
  • flex-wrap:example
  • justify-content:example
  • align-items:example
  • flex-grow:#
  • flex-shrink:#

CSS Transformations

  • transform:rotate(deg)
  • transform:scale(#)
  • transform:skew(deg)

CSS Resets

  • separate file to make all browsers start at the same place with your styles
  • makes all elements style the same across all browsers

CSS Pseudos

  • example:hover
  • example:active
  • example:focus
  • example::before
  • example::after
  • example::nth-child
  • example::first-child
  • example + example
  • example > *

CSS Variables

  • --example-variable
  • var(--example-variable);
